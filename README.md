## Název Projektu

Arc Raiders

## Žák

Nicolas Doktor

## Stručný popis tématu 

Webová prezentace zaměřená na hru ARC Raiders a její herní svět.
Stránka bude mít podobu one-page webu s několika sekcemi, které představí základní lore, herní mechaniky, dostupné třídy a aktuální novinky. 
Cílem projektu je vytvořit moderní, přehledný a responzivní web zaměřený na prezentaci hry.

## Cílová skupina

Hráči zajímající se o ARC Raiders, fanoušci sci-fi kooperativních akcí a návštěvníci hledající informace o hře, mechanikách a herním obsahu.

## Návrh obsahových sekcí

O hře – základní představení ARC Raiders, příběhové pozadí a atmosféra světa napadeného Roboty.

Herní mechaniky – popis gameplaye, kooperace, typy misí a soubojového systému.

Třídy a vybavení – dynamicky načítaný seznam herních tříd, zbraní a vybavení s popisy a vizuály.

Novinky a aktualizace – informace o vývojových novinkách, patch notes a oznámeních.

Kontakt / Komunita – odkazy na oficiální kanály, Discord, sociální sítě a možnost odeslat dotaz přes jednoduchý formulář.

## Návrh designu / wireframe

Barevné ladění v futuristických a industriálních tónech.

Přehledné blokové uspořádání sekcí pod sebou s výraznými titulky inspirovanými sci-fi estetikou.

Jednoduché a čitelné UI s moderními prvky.

## Použité technologie

Frontend: HTML, CSS nebo Tailwind CSS, JavaScript

Data: JSON

AJAX: načítání obsahu sekcí

Backend: jednoduchý Python

Verzování: Git + GitHub
